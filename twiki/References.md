## NIPSCERN / UFJF

## Technologies used

- [Three.js](https://threejs.org) — WebGL renderer powering the viewer.
- [ROOT](https://root.cern) — source format for `CaloGeometry.root`.
- [Rust + wasm-pack](https://rustwasm.github.io/wasm-pack/) — toolchain for the ATLAS-ID parser.
- [glTF 2.0 / GLB](https://www.khronos.org/gltf/) — on-wire 3-D format.
