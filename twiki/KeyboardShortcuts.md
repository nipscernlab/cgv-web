# Keyboard Shortcuts

Single-key shortcuts, case-insensitive. Ignored while typing in an input
(e.g. a threshold field).

| Key       | Action                                                  |
|-----------|---------------------------------------------------------|
| **M**     | Toggle the left sidebar (menu).                        |
| **E**     | Toggle the right-side [energy threshold](EnergyThresholds.md) panel. |
| **G**     | Toggle all [ghost envelopes](Geometry.md#ghost-envelopes). If any is on, turns everything off; else restores the default TileCal set. |
| **B**     | Toggle the [beam axis](Geometry.md#beam-axis).         |
| **I**     | Toggle hover tooltip / cell info.                      |
| **R**     | Reset the camera to the default position.              |
| **C**     | Enter **cinema mode** (hide UI, auto-rotate).          |
| **P**     | Open the screenshot dialog.                            |
| **S**     | Open the Settings popover.                             |
| **T**     | Toggle the **TILE** detector layer.                    |
| **L** / **A** | Toggle the **LAr** detector layer.                 |
| **H**     | Toggle the **HEC** detector layer.                     |
| **J**     | Toggle particle **tracks**.                            |
| **K**     | Toggle **clusters**.                                   |
| **Esc**   | Close the front-most popover / exit cinema mode.       |

## Mouse

| Gesture                    | Action        |
|----------------------------|----------------|
| Left-drag                  | Rotate camera |
| Right-drag (or Ctrl+left)  | Pan camera    |
| Wheel                      | Zoom          |
| Hover over cell            | Show tooltip (if cell-info is on) |
| Click outside a popover    | Close it      |

## Touch (mobile / tablet)

One finger rotates, two fingers pan, pinch zooms. The toolbar collapses to
a **Toolbar** pill you can tap to re-open.
