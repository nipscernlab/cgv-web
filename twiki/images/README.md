# TWiki Images

Drop screenshots and diagrams in this folder, then reference them from any
TWiki page with standard Markdown image syntax:

```markdown
![caption](images/filename.png)
```

## Suggested captures

| File                         | What to capture                                          |
|------------------------------|----------------------------------------------------------|
| `viewer-overview.png`        | Full viewer, event loaded, sidebar + toolbar visible     |
| `live-mode.png`              | Live panel with an active ATLANTIS feed                  |
| `local-mode.png`             | Local upload panel, one XML selected, carousel ready     |
| `samples-mode.png`           | Samples list loaded from `default_xml/index.json`        |
| `toolbar.png`                | Bottom toolbar, each button labelled                     |
| `tile-threshold.png`         | Energy threshold panel open on the TILE tab              |
| `cluster-panel.png`          | Cluster tab with threshold slider                        |
| `ghost-panel.png`            | Ghost popover showing the nine envelope toggles          |
| `layers-panel.png`           | Detector layers popover (TILE / LAr / HEC)               |
| `settings-panel.png`         | Settings: preferences, shortcuts, sponsors               |
| `cinema-mode.png`            | Cinema mode with auto-rotation and hidden UI             |
| `screenshot-dialog.png`      | Screenshot resolution picker                             |

Keep images ≤ 1 MB (PNG 85 % quality is usually enough) so the repo stays
lightweight.
